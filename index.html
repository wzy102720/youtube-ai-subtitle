YouTube 双语字幕

拦截 YouTube 字幕请求，预先翻译，叠加显示中英双语字幕。

一键安装脚本

点击上方按钮，油猴会自动弹出安装对话框。 还没装油猴？先装 Tampermonkey。

劫持原生字幕直接拦截 YouTube 的 /api/timedtext 请求
智能切分只在标点处切，不把一句话切到一半
双引擎DeepSeek 质量高 / Google 免费
本地缓存每个视频只翻译一次

快速上手

  1. 装 Tampermonkey 浏览器扩展
  2. 点上方"一键安装脚本"按钮，确认安装
  3. 到 platform.deepseek.com 拿一个 DeepSeek API Key（充值 10 元能用很久）
  4. 打开任意 YouTube 视频 → 油猴菜单 → 设置 DeepSeek API Key
  5. 必须点开 YouTube 自己的 CC 字幕按钮，脚本要靠它发字幕请求才能拦截

没 DeepSeek Key 怎么办

切到 Google 翻译就行，免费、不需要 Key：

  • 油猴菜单 → 切换引擎 (当前: deepseek) → 改成 google
  • 刷新页面

翻译质量比 DeepSeek 差一些，但完全够用。
