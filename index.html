YouTube Bilingual Subtitles

Intercept YouTube's caption requests, translate them ahead of time, and overlay bilingual subtitles on the video.

Install Script

Click the button above and Tampermonkey will automatically prompt to install. Don't have Tampermonkey yet? Install Tampermonkey first.

Hijacks native captionsIntercepts YouTube's /api/timedtext requests at the network layer
Smart splittingBreaks only at punctuation, never mid-sentence
Two enginesDeepSeek (quality) or Google (free)
Local cacheEach video is translated once, replays read from cache

Note: This script translates English captions into Chinese. The Tampermonkey menu labels and the in-page debug banner are in Chinese.

Quick start

  1. Install the Tampermonkey browser extension
  2. Click the "Install Script" button above and confirm
  3. Get a DeepSeek API key at platform.deepseek.com (top up ¥10 — that lasts a long time)
  4. Open any YouTube video → Tampermonkey menu → 设置 DeepSeek API Key (Set DeepSeek API Key)
  5. Turn on YouTube's CC subtitle button — the script intercepts YouTube's caption requests, so CC must be active

No DeepSeek key?

Switch to Google Translate — free, no key required:

  • Tampermonkey menu → 切换引擎 (当前: deepseek) ("Switch engine") → choose google
  • Reload the page

Translation quality is a little worse than DeepSeek, but perfectly usable.
